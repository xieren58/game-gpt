## 使用
一共有3种推荐方式，后2种需要登录Steam账号.
- 随机推荐
- 从库存推荐（从你的库存中推荐一款游戏）
- 根据游玩时长推荐（根据你库存中游玩时间最长的几款游戏，推荐一款新游戏）
